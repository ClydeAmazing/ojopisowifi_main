# OJO PISOWIFI
Simple, Easy to use full-fledged Wifi Vending Solution

### Key features:
1. Time based wifi vending
2. Paperless
3. Wifi Vouchers
4. Portal Customization
5. Simple Admin Dashboard
6. Open Source
7. Raspberry Pi 3 Compatible
8. Orange Pi One Compatible
